RepoToLLM 🧠💻

RepoToLLM is a powerful VSCode extension designed to bridge your repository with AI tools like ChatGPT, Claude, and Gemini. With this tool, you can export your repository into a single Markdown file that provides:

  • 🌳 A detailed repository structure overview.
  • 💻 A list of all files with their respective paths.
  • 📂 Complete code content for each file.

The generated Markdown file can be directly copied into LLM tools to:

  • 🤖 Get quick insights into your repository.
  • ⚡ Accelerate your development cycles.
  • 🐛 Fix bugs faster with contextual assistance.

Usage:
Open your Python project in VSCode and run the command Repo To LLM. This generates a REPO_TO_LLM.md file in your repository root, which you can paste into AI tools for enhanced productivity.
