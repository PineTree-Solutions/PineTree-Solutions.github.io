RepoToLLM

RepoToLLM is a powerful VSCode extension that bridges your repository with AI tools. It generates a detailed Markdown file containing:

  • A comprehensive repository structure overview
  • A list of all files with their respective paths
  • Complete code content for each file

Use the generated Markdown file to accelerate development cycles, fix bugs faster, and gain valuable insights into your repository.

Usage:
Open your Python project in VSCode and run the command Repo To LLM. The generated REPO_TO_LLM.md file is ready to be used with AI tools.
